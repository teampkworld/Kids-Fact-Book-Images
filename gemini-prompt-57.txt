**Add Bullet Points to the Existing TABLE OF CONTENTS Page**

Enhance the existing **TABLE OF CONTENTS** page by adding a bullet point before every Table of Contents entry. Do **not** change the current page structure, layout, typography, page numbering, borders, margins, spacing, or leader dots. Only add bullet points while preserving the existing design.

### Requirements

* Add **one bullet point** before every Table of Contents heading.
* All bullet points must:

  * Be identical in size, shape, weight, and color.
  * Be perfectly aligned vertically and horizontally.
  * Maintain equal spacing between the bullet and the heading text.
  * Be centered relative to the text baseline.
  * Use a consistent left margin throughout the page.
* Ensure every Table of Contents entry starts at the same horizontal position after the bullet.
* The bullet points must not overlap with icons, text, leader dots, or page numbers.
* Maintain uniform spacing between:

  * Bullet → Heading text
  * Heading text → Leader dots
  * Leader dots → Page numbers
* If the Table of Contents spans multiple pages, apply the same bullet style, alignment, and spacing consistently on every page.
* Ensure the bullets remain sharp and high-quality in the exported PDF without pixelation or distortion.
* Keep the page fully compliant with Amazon KDP print standards, including safe margins and bleed.
* Do **not** modify any other content, fonts, icons, page references, headings, or layout elements. Only add the bullet points while preserving the existing design exactly.

## 3. Automatic Pagination for Professional Style Presets

The application currently includes **112 Professional Style Presets**.

When the user uploads one or more SVG files, the application must apply **every Professional Style Preset exactly once**, without repetition.

For example, if the user selects:

* **Vertical = 5**
* **Horizontal = 6**

then:

* Each collage page contains **30 unique preset variations** (5 × 6 = 30).

The system should automatically distribute all 112 presets across multiple collage pages:

* **Page 1:** Presets 1–30 (30 unique presets)
* **Page 2:** Presets 31–60 (30 unique presets)
* **Page 3:** Presets 61–90 (30 unique presets)
* **Page 4:** Presets 91–112 (22 unique presets)

No preset should appear more than once until every preset has been used.

## 4. Randomized Unique Preset Assignment

If the user enables **Random Preset Order**, the application should:

* Randomly shuffle the complete list of available presets.
* Apply each preset only once.
* Never repeat a preset during the current batch generation.
* Continue assigning unused presets until all presets have been exhausted.

For example:

If 112 presets are available:

* Randomly shuffle all 112 presets.
* Apply the first 30 unique presets to Page 1.
* Apply the next 30 unique presets to Page 2.
* Apply the next 30 unique presets to Page 3.
* Apply the remaining 22 unique presets to Page 4.

Only after all 112 presets have been used should the application allow a new randomization cycle.

## Adjust Spacing Between the Fact Title Badge and the Icon (31 Grid Pages)

Use the attached reference image as the design reference.

**Reference Image:**
`https://raw.githubusercontent.com/teampkworld/Kids-Fact-Book-Images/main/Unti456tled.jpg`

---

# Current Issue

On the **31 Image Fact grid pages**, the **Fact Title** (displayed inside the black rounded rectangle) is positioned too close to the category icon.

As a result:

* The rounded title badge and the icon appear to be touching or merging.
* The layout looks crowded.
* There is insufficient visual separation between the two elements.

---

# Required Fix

Add a **small, consistent horizontal gap** between the **Fact Title badge** and the **icon**.

The spacing should be just enough to create clear visual separation while maintaining a balanced, professional layout.

---

# Layout Requirements

Only adjust the spacing between these two elements:

* **Fact Title Badge**
* **Category Icon**

Do **not** modify:

* Title position
* Icon position
* Badge size
* Font size
* Icon size
* Grid layout
* Image position
* Description position
* Page margins
* Overall alignment

The only change should be the addition of a small horizontal space between the badge and the icon.

---

# Spacing Rules

* Maintain a consistent gap for all 31 grid pages.
* The gap should scale correctly for both short and long titles.
* Prevent the badge and icon from touching or overlapping.
* Preserve the current vertical alignment of both elements.

---

# Responsive Behavior

If the title length changes:

* Keep the icon aligned in its current position.
* Automatically maintain the same minimum gap between the title badge and the icon.
* Ensure the spacing remains visually consistent across all 31 fact cards.

---

# Validation Checklist

Before generating the final PDF, verify that:

* ✅ Every one of the 31 fact title badges has a small, consistent gap from its corresponding icon.
* ✅ The badge and icon no longer appear merged.
* ✅ No overlap occurs, regardless of title length.
* ✅ No other page elements move or change position.
* ✅ The grid layout remains exactly the same.
* ✅ The final appearance matches the reference image and looks clean, balanced, and professional.

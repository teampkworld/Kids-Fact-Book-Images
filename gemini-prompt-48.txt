---

## Step 1 – Remove the Existing Workflow

Completely remove the current **"MAP OF YOUR ADVENTURE"** icon workflow.

Delete all existing:

* Hardcoded icon mappings
* Default icon arrays
* Previous SVG upload logic
* Old icon lookup functions
* Cached SVG references
* Memory references
* Local storage/session storage entries
* Object URLs
* Fallback icon logic
* Previous initialization code

After this cleanup, the application should have **no remaining knowledge** of the previous icon system.

---

# Step 2 – Create a New Default Icon Library

Instead of uploading icons or loading bundled assets, create a **new default icon library** using the GitHub Raw SVG URLs.

The GitHub URLs become the **single source of truth**.

The application must never search elsewhere.

---

# Step 3 – Create a Dictionary (Key → URL)

Do **not** use arrays or indexes.

Create a dictionary/object where:

```
Heading Name
↓

SVG URL
```

Example

```
WELCOME TO WONDER!
↓

https://raw.githubusercontent.com/teampkworld/Files/main/Index-Icons/Waving%20Hand.svg
```

Every heading must have exactly one URL.

Never rely on array position.

Never rely on upload order.

Never rely on image numbering.


**Collage Preview & Exports – Remove All Text Labels & Use Solid White Background**

In the **Collage Preview & Exports** section, update the rendering logic for both the preview and all exported collage images.

### Required Changes

1. **Do not render any text below or above the images.**

   * Remove the original SVG filename (e.g., `Technology.svg`).
   * Remove the applied preset name (e.g., `Vintage Velvet`, `Summer Solstice`, `Warm Flame`, `Orange Burst`).
   * Remove every other label, caption, filename, watermark, identifier, or metadata text from the collage.
   * The collage should contain **only the rendered images**.

2. **Export a clean photo collage only.**

   * Every collage cell should contain only the recolored SVG artwork.
   * No text, no filenames, no preset names, no numbering, and no extra UI elements should appear in the exported PNG or SVG collage.

3. **Use a solid white background.**

   * Replace the current transparent/checkerboard background with a **pure white (#FFFFFF) background**.
   * The white background must cover the **entire collage canvas**, including all margins, spacing, and empty areas.
   * The checkerboard transparency pattern should never appear in either the preview or exported files.

4. **Preserve image quality.**

   * Keep all SVGs perfectly centered within each collage cell.
   * Maintain the original SVG quality, fill, stroke, gradients, transparency (inside the artwork only), proportions, and sharp vector edges.
   * Do not crop, stretch, distort, or resize the artwork incorrectly.

5. **Apply this to every output format.**

   * Preview
   * PNG Photo Collage Export
   * SVG Photo Collage Export
   * Print-Quality Export
   * ZIP Batch Export
   * Future export formats

### Validation Rules

Before generating or exporting any collage, automatically verify that:

* ✅ No SVG filename is displayed.
* ✅ No preset name is displayed.
* ✅ No captions or labels are displayed.
* ✅ No watermark or metadata text is displayed.
* ✅ The collage background is **100% solid white (#FFFFFF)**.
* ✅ All SVG artworks remain perfectly centered and aligned.
* ✅ Image quality remains sharp and identical to the original rendered SVG.
* ✅ The exported collage contains **only the artwork arranged in the selected grid**, with no additional text or UI elements.
